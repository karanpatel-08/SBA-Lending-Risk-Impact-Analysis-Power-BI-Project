# 📌 Project Overview

This project focuses on analyzing SBA (Small Business Administration) lending data to understand loan distribution, default risk patterns, lender performance, and economic impact using Power BI.

The objective was to transform raw lending data into interactive business intelligence dashboards capable of supporting risk assessment, portfolio monitoring, and strategic lending analysis.

The dashboards were designed from a financial analytics perspective to help stakeholders monitor lending performance, identify high-risk segments, and evaluate the broader economic contribution of SBA-backed loans.

---

# 🎯 Business Problem

Financial institutions and lending agencies face several challenges while managing SBA loan portfolios, including:

- Rising loan default risks
- Limited visibility into lender performance
- Difficulty identifying high-risk industries and regions
- Challenges in monitoring loan approval trends
- Lack of centralized lending analytics

Without proper analytics, lenders may struggle to optimize lending strategies, monitor portfolio risk, and evaluate business impact effectively.

This project aims to address these challenges through interactive Power BI reporting and lending intelligence analysis.
